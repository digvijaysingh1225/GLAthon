WELCOME TO STUDENT
RECOMMENDATION SYSTEM

The software assists the university in automating the selection of the viable candidates for the process of recuitment.
Our vision is to automate the process of shortlisting by the usage of Machine Learning and Data Analysis. We provide various improvement solutions to recruiters to enhance retention rates of students using various demographics and analyzing latest technological trends.


Let's take a deep dive into what are the key features and highlights which make this project a useful and a handy tool for any recruitment process.


HIGHLIGHTS/ KEY FEATURES


Selecting viable candidates for a particular company using recommendation system model.
Giving a detailed visualization of students preferred programming language, CP platform engagement, Leetcode progress and other co-curricular activities.
Using earlier placed student's to data to generate insights to help retention strategies.
